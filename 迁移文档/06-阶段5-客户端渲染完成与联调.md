| 2e | ✅ | 工具提示 — StorageGridView hover 物品手动渲染 tooltip（绕过 GuiScreen.renderToolTip protected 限制） |

**拼音搜索软兼容方案（最终架构）**：

```
客户端安装了 NEC（含 PinIn）
  → StorageViewModel.applySearch() → RtsPinyinSearch.matches()
  → 反射 PinIn.contains() 做拼音匹配
  → 搜索 "ts" 可匹配 "铁锭"（拼音首字母）

客户端未安装 NEC
  → RtsPinyinSearch.matches() 降级为 source.contains(query)
  → 搜索 "iron" 可匹配 "minecraft:iron_ingot"

服务端（始终）
  → RtsStorageSession.matchesSearch() 仅做 itemId.toLowerCase().contains(query)
  → 零外部依赖，无需安装任何拼音 mod
```

**涉及文件**：
| 文件 | 修改 |
|------|------|
| `util/RtsPinyinSearch.java` | 静态单例 PinIn 引擎，`matches(source, query)` → boolean，`isAvailable()` → boolean |
| `server/storage/RtsStorageSession.java` | `matchesSearch()` → 纯 `itemId.contains(query)`（移除 PinIn 反射） |
| `client/StorageViewModel.java` | 新增 `filteredEntries`、`applySearch(query)`、`getDisplayEntries()`、`clearSearch()` |
| `client/panel/storage/StorageGridView.java` | 搜索栏渲染 + onKeyTyped 处理 + getDisplayEntries() 渲染 + 手动 tooltip |
