## 1. 项目简介

RTSbuilding ("RTS Building: Build From Above") 是一个 **RTS 风格的俯视建造模组**，允许玩家以鸟瞰视角进行远程放置/破坏方块、链接存储容器、管理建造项目（蓝图）以及技能树解锁。原项目基于 **Minecraft 1.21.1 NeoForge**，由 JerryLunar 与 H-crab 开发；当前项目已完整移植至 **Minecraft 1.7.10 Forge**（GTNH 生态），源码约 **120 个 Java 文件**。

### 核心功能
| 功能 | 说明 |
|------|------|
| 俯视相机 | WASD 自由移动 + 缩放，实体 `RtsCameraEntity` |
| 远程建造 | 放置/破坏/交互/旋转方块、批量放置、流体操作 |
| 存储链接 | 链接箱子等容器，统一浏览/存取/排序/漏斗 |
| 合成终端 | 远程合成、JEI/NEI 配方转移 |
| 蓝图系统 | 多格式导入（Litematica, SpongeSchem, Building Gadgets, NBT）、放置 |
| 技能树 | 生存模式渐进解锁，18 个技能节点 |
| 采矿 | 连锁挖掘 (Ultimine)、区域采矿/破坏 |
| AE2 兼容 | ME 网络存储接入 |
| NEI 兼容 | 配方覆盖层、Shift+Click 转移 |

## 6. 注册表操作

| 操作 | 位置 | 说明 |
|------|------|------|
| 网络通道 | `RtsNetworkManager.CHANNEL = "rtsbuilding"` | SimpleNetworkWrapper |
| 实体注册 | `CommonProxy.preInit()` | `RtsCameraEntity`，trackingRange=128 |
| 按键绑定 | `ClientProxy.preInit()` | 25 个 KeyBinding（`key.rtsbuilding.*`） |

## 9. 关键数据

- **源码文件总数**: ~120 个 `.java` 文件
- **网络消息数**: ~63 条（discriminators 0-60+）
- **技能节点数**: 18 个（`RtsProgressionNodes`）
- **客户端面板数**: 10 个（顶栏 + 存储网格 ×3 + 合成 + 弹窗 ×2 + 工具槽 + 蓝图）
- **世界渲染器**: 6 个（形状预览、边界、目标、动画、存储高亮、破碎进度）
- **国际化语言**: 4 种（en_us, zh_cn, zh_hk, zh_tw），各 2 种格式（.lang + .json）
- **按键绑定**: 25 个
- **构造模式**: 6 种（OFF, SELECT_PAN, LINK_STORAGE, FUNNEL, INTERACT, ROTATE）
