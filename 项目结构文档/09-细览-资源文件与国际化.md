## 1. 资源文件概览

RTSbuilding 的资源文件位于 `src/main/resources/`，包含 Mod 元信息、Mixin 配置、语言文件和 GUI 纹理。

### 顶层资源

| 文件 | 用途 |
|------|------|
| `mcmod.info` | Forge Mod 元信息描述文件 |
| `pack.mcmeta` | 资源包元数据 |
| `mixins.rtsbuilding.json` | Mixin 配置（运行时加载） |
| `rtsbuilding.mixins.json` | Mixin 配置副本 |

## 3. 国际化 (i18n)

### 3.1 语言文件结构

```
assets/rtsbuilding/
├── lang/                          ← Forge 1.7.10 .lang 格式
│   ├── en_us.lang
│   ├── zh_cn.lang
│   ├── zh_hk.lang
│   └── zh_tw.lang
│
└── rtsbuilding/lang/              ← 新版 .json 格式（供迁移兼容参考）
    ├── en_us.json
    ├── zh_cn.json
    ├── zh_hk.json
    └── zh_tw.json
```

### 3.2 支持的语言

| 语言 | .lang | .json | 用途 |
|------|-------|-------|------|
| 英语 (US) | `en_us.lang` | `en_us.json` | 默认语言 |
| 简体中文 | `zh_cn.lang` | `zh_cn.json` | 中国大陆 |
| 繁体中文 (HK) | `zh_hk.lang` | `zh_hk.json` | 香港 |
| 繁体中文 (TW) | `zh_tw.lang` | `zh_tw.json` | 台湾 |

### 3.3 .lang 格式 (Forge 1.7.10)

```
# 键=值 格式
key.rtsbuilding.toggle_rts=Toggle RTS Mode
key.rtsbuilding.open_screen=Open RTS Screen
key.rtsbuilding.camera_forward=Camera Forward
...
```

```

│   ├── box_block.png

└── topbar/                 ★ 顶部工具栏 (9 个功能 × 4 个状态 = 36 个 PNG)
    ├── chunk_view_{state}.png        # 区块视图 (active/hover/inactive/pressed)
    ├── mode_funnel_{state}.png       # 漏斗模式
    ├── mode_interact_{state}.png     # 交互模式
    ├── mode_link_{state}.png         # 链接模式
    ├── mode_rotate_{state}.png       # 旋转模式
    ├── quest_detect_{state}.png      # 任务探测
    ├── quick_build_{state}.png       # 快速建造
    ├── settings_gear_{state}.png     # 设置
    └── ultimine_{state}.png          # 连锁挖掘
```

### 4.2 纹理状态

每个 topbar 图标有 4 个状态：

| 状态 | 文件名后缀 | 触发条件 |
|------|-----------|---------|
| Inactive | `_inactive.png` | 正常显示 |
| Hover | `_hover.png` | 鼠标悬停 |
| Active | `_active.png` | 功能激活中 |
| Pressed | `_pressed.png` | 鼠标按下 |

## 7. 资源加载机制

Forge 1.7.10 资源加载路径：

```
src/main/resources/assets/rtsbuilding/
    ↓ 编译打包
rtsbuilding-{version}.jar!assets/rtsbuilding/
    ↓ 运行时
Minecraft.getMinecraft().getResourceManager()
    ↓
ResourceLocation("rtsbuilding", "textures/gui/...")
```

语言文件加载路径：

```
assets/rtsbuilding/lang/{lang_code}.lang
    ↓
I18n.format("key.rtsbuilding.toggle_rts")
    → 返回对应语言的翻译字符串
```
